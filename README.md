# 📊 Sales Operations Dashboard

## Overview

This project analyzes a sales dataset using Google Sheets.

The objective was to clean inconsistent business data, perform exploratory analysis using Pivot Tables, and build an executive dashboard highlighting key business metrics.

## Features

- Data Cleaning
  - Removed leading/trailing spaces
  - Standardized text values
  - Parsed multiple date formats
  - Flagged invalid dates
  - Identified missing revenue values

- KPI Dashboard
  - Total Revenue
  - Total Orders
  - Total Customers
  - Missing Revenue Records

- Pivot Table Analysis
  - Revenue by Region
  - Revenue by Customer Size
  - Monthly Revenue Trend
  - Revenue by Country

---

## Key Insights

- Total Revenue: **$4.86M**
- Enterprise customers generated approximately **73%** of total revenue.
- EMEA was the highest revenue-generating region.
- United States contributed the highest revenue among all countries.
- February recorded the highest monthly revenue.
- 22 records contained invalid dates and were excluded from monthly trend analysis.
